## üöÄ Getting Started

### Prerequisites
*   **Python 3.8+** installed on your machine.
*   **pip** package manager.

### üì• Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/ankush850/Python_project-.git
    cd Python_project
    ```

2.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```
    > üí° **Note:** `requirements.txt` contains libraries for *all* projects. For a specific script, you can install only what you need (e.g., `pip install requests` for the weather app).
